# Projet Poudelard — L’Art de Coder comme un Sorcier

**TI101 – Programmation en Python**

## Description
Jeu d’aventure interactif en Python inspiré de l’univers de Poudlard.  
Le joueur crée son personnage, traverse plusieurs chapitres narratifs et participe à des mini-jeux (quiz, sorts, Quidditch).

## Prérequis
- Python 3.8 ou plus
- Git installé

## Installation
```bash
git clone https://github.com/remichoiselat20-ai/Projet_Poudelard.git
cd Projet_Poudelard
```

## Lancement du jeu
```bash
python main.py
```

## Structure du projet
```
Projet_Poudelard/
│
├── main.py
├── menu.py
├── chapitres/
├── univers/
├── utils/
└── data/
```

## Fonctionnalités
- Création de personnage
- Répartition dans une maison
- Apprentissage de sorts
- Quiz magique
- Match de Quidditch
